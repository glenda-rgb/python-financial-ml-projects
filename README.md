# Python Financial Machine Learning Projects

This repository contains simple, self-driven Python machine learning projects using synthetic financial data.

## Projects

### 1. Financial Transaction Classification
- Built a supervised machine learning model to classify financial transactions.
- Used Python, pandas, and scikit-learn.
- Demonstrates data preprocessing, model training, and evaluation.

### 2. Financial Expense Trend Prediction
- Developed a regression model to predict monthly expenses.
- Applied linear regression for basic financial forecasting.
- Demonstrates independent data analysis and modeling skills.

## Tools & Technologies
- Python
- pandas
- scikit-learn

## Notes
All data used in these projects is synthetic and created for learning and demonstration purposes.
